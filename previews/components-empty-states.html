Empty states are intentionally minimal — a centred sentence in colorScheme.onSurfaceVariant with optional Bebas Neue heading. No illustrations, no hand-drawn icons, no exclamation marks. Title + body pattern is reserved for "blocking" empties (no match this weekend, season complete, filler hub) where the user should know what's next. Single-line copy is enough for list emptiness (news.no_articles, videos.no_videos). All copy lives in app_strings_defaults.dart and is CMS-overridable.
