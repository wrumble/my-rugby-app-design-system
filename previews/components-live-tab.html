Live tab widgets are the bulk of match_hub/presentation/widgets/live/. All sit on Match Hub navy surface via AppCard + the local SectionHeader. MomentumBar is built on the shared DualProgressBar primitive (auto-normalises so home + away ≤ 1, animates with AppDurations.slow + AppCurves.standard). EventFeedSection uses an emoji-icon convention (🏉 try, ✅ conversion, 🎯 penalty, 🟨 yellow, 🟥 red, ⬇️ scrum, 📌 lineout) — try events get a 3pt left border in colorScheme.primary as a colour+structure cue. HtQuizWidget shows during HALFTIME phase only; uses FilledButton pill submit. Initial event feed shows 5; AnimatedSize reveals the rest with AppCurves.decelerate on tap of "Show more events".
