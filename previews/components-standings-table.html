Standings is the canonical data-table pattern for any tabular data in the app. Fixed numeric column widths (PL=36, PD=48, Pts=40) sized for the narrowest phone viewport — team name is the only flexible column. League name is rendered with the same Bebas Neue display-sm style as AppSection headings. Header row uses colorScheme.surfaceContainerHighest with labelSmall + 0.5px letter-spacing. Numeric cells use AppTypography.monoSm. The user's selected team row is tinted with colorScheme.primaryContainer and bumped to FontWeight.bold. Sort: totalPoints desc, then pointsDifference desc. Pool tab bar appears only when there are 2+ pools; league dropdown appears only when there are 2+ leagues. Use this pattern for any future leaderboard/stats grids.
